# FIRE-002 — Fire Compartmentation Handover & Validation Sheet

Interactive engineering handover for automating QA/QC check **FIRE-002 — Fire
compartmentation plans complete** (NCC 2022 Part C3 / former Table C2.2,
AS 1530.4, DBP Act 2020 NSW).

Written for a software engineer with zero building knowledge: the check
decomposed into four decidable sub-checks, drawing references and code basis
per ask, copyable validation prompts, a worked run against a real 9-sheet
fire series, ranked traps, and reviewer comment boxes (saved in the browser,
exportable as JSON).
