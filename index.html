Q1

Which jurisdiction and code edition — any local amendments?

CONFIRMED FROM SETNSW, Australia · NCC 2022 (NSW variations) · DBP Act 2020 overlay

Jurisdiction: New South Wales, Australia. Four independent signals on the sheets agree: the project address (2C–2D Wolseley Street, Drummoyne — Sydney), the consent number in NSW DA format (DA2022/0164, MOD 2024/0036), a Regulated Design Record block with DEP practitioner numbers (exists only under NSW's DBP Act), and the architect entity "SJB Architecture (NSW) Pty Ltd", Surry Hills NSW.

Code edition: NCC 2022. Construction Certificate issues run Jan 2024 → Aug 2025, well after NCC 2022 adoption (1 May 2023 in NSW) — so NCC 2022 with NSW state variations governs. Which exposes a mis-citation in the check itself: "NCC 2022 Part C2, Table C2.2" doesn't exist — 2022 renumbered compartmentation to Part C3, Table C3D3 (limits unchanged). "C2.2" is NCC 2019 numbering. The tool must alias both and cite both, because half the industry still says C2.2.

Local amendments that matter: NSW state variations to the NCC (incl. an NSW-specific C3D6), and the DBP Act 2020 / Regulation 2021 — fire compartmentation is a "regulated design" for any building with a Class 2 part, so every fire sheet must carry a declared practitioner. That's an NSW-only proxy: skip it in other states, don't fail its absence.

Q2

What stage is the client's set — CD? Are we enforcing or advising?

NUANCE FOUNDFiled as "CD Set" — but the title blocks say otherwise

The set is past CD. The file is named "CD Set", but every title block reads FOR CONSTRUCTION under a CONSTRUCTION CERTIFICATE staging regime — the register legend maps revisions to CC packages (CC1 excavation & shoring → CC4 remainder of works), and revisions A through O (Jan 2024–Aug 2025) are all "For Construction" issues. This is an issued IFC/CC set, not a design-development snapshot.

Enforcing vs advising follows from that. FIRE-002 is scoped to DD and CD — its natural home is a gate: severity Critical means a FAIL blocks the set from issuing. Run against an already-issued FC set, the same check becomes an audit: findings are advisory, routed to the architect and certifier for the next revision, because the tool has no issue to block. The tool's verdicts don't change — only who acts on them and how fast. The output should therefore carry a mode: enforce | advise flag derived from the title-block status, never hard-coded.

Fire compartmentation of a building with a Class 2 part is a regulated design: a registered practitioner must prepare and declare it before building work — so the plans must exist as declared documents, per level, at the current revision. C3D3(2) is why a roof-only plant level is legitimately exempt from needing one.

002.2

Boundaries shown and labelled — drawings & code basis

Drawings

Code basis

A compartment boundary is only a boundary if it's a rated construction — so every drawn boundary must carry an FRL, and every FRL number traces to an AS 1530.4 furnace test. The tool checks notation exists and is well-formed (values in {30, 60, 90, 120, 180, 240}); it never re-derives the rating.

002.4

Areas annotated — drawings & code basis

Drawings
AR-2101–2106 · pp.5–10 (in-plan area text — none found) Any compartment schedule table in the series (none exists)
Code basis
DBP Act 2020 s.9 compliance declaration NCC 2022 C3D3 (auditability of the limit)

No NCC clause literally says "write the m² on the sheet" — the requirement is evidentiary. The declared design must let the certifier verify the C3D3 limits without re-deriving them; an undeclared area makes the regulated design unauditable. That's why this sub-check is documentation-completeness, and why its FAIL here (zero m² tokens across pp.5–13) is exactly the omission FIRE-002 exists to catch.

Validation prompt — 002.4
Sweep every 2100-series plan sheet for area tokens matching
[\d,\.]+\s*(m²|m2|sqm) — e.g. "1,234 m²", "1234m2", "AREA: 1,234 SQM".
Associate each token to a compartment by centroid-inside-region, leader line,
or a schedule row keyed by the 002.2 compartment IDs (accept tabulated
schedules as equivalent to in-plan annotation). Every compartment needs
exactly one declared area: zero = FAIL that compartment naming its sheet;
conflicting values = REVIEW. On this set the sweep returns zero hits on all
nine sheets → 002.4 FAIL, driving the overall Critical verdict.
